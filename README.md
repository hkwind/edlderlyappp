# 腦友所依

「腦友所依」 is a mobile-first prototype for family-only dementia and elderly care support. The prototype focuses on reducing cognitive load for the elder user while giving caregivers a clear way to configure destinations, medicine reminders, and custom quick actions.

## Tech

- Vite
- React
- TypeScript
- React Router
- Zustand
- Mock data only

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`

## Prototype scope

- Elder 6-grid home
- Return home flow
- Other places flow
- Call family flow
- Medicine flow
- Two custom slots
- Caregiver dashboard
- Mock location and battery status

No backend, no auth, no real notifications, no real GPS, and no cloud sync are included.
