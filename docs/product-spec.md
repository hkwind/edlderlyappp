# Product Spec

## Product goal

Build a family-only prototype that helps an elder user complete essential daily actions with less confusion, while giving a caregiver an easy way to configure that experience.

## Primary users

### Elder user

- May forget destinations
- May forget medicine
- Needs large targets and short labels
- Needs calm, obvious navigation

### Caregiver

- Configures the elder-facing experience
- Reviews reminders, destinations, and status
- Needs low-friction editing

## Core principles

- Reduce cognitive load
- Keep all core elder actions on the home screen
- Use short, concrete labels
- Prefer obvious routes over dense interfaces
- Mock first, no production integrations

## Prototype features

- Elder 6-grid home
- Return Home screen
- Other Places screen
- Call screen
- Medicine list for today
- Two configurable custom slots
- Caregiver dashboard
- Caregiver editing for destinations, medicine, and custom slots
- Mock location and battery panels
