# User Flows

## Elder: use a saved destination

1. Open elder home
2. Tap `去其他地方`
3. Pick a saved place
4. View a simple destination instruction card

## Elder: return home

1. Open elder home
2. Tap `回家`
3. View home destination details and clear next step text

## Elder: call family

1. Open elder home
2. Tap `打電話`
3. Tap one large contact button
4. See mock call confirmation

## Elder: take medicine

1. Open elder home
2. Tap `食藥`
3. See today’s medicine tasks
4. Recognize `即將到來`, `已完成`, or `錯過`

## Caregiver: update the elder setup

1. Open caregiver dashboard
2. Review destination, medicine, and status cards
3. Enter the relevant edit screen
4. Save the mock configuration in local state
